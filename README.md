🏨 Hospitality Domain Analysis – Notebook Overview
In this project, I analyzed a hospitality dataset to uncover key insights about customer behavior and business performance.

🔍 Key Steps & Highlights:

📥 Data Loading and Inspection

Loaded the CSV file using pandas.

Explored the dataset with .head(), .info(), .describe().

🧹 Data Cleaning

Checked for missing values and duplicates.

Handled categorical data and corrected data types if needed.

📊 Exploratory Data Analysis (EDA)

Used Seaborn and Matplotlib to create visualizations such as:

Booking distribution

Revenue by market segment

Cancellation patterns

Guest preferences

🎯 Business Insights

Identified trends like peak booking months, customer types with higher cancellations, and factors influencing revenue.

These insights can help hospitality businesses improve marketing, retention, and forecasting.
